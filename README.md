# A2liX Demo

Symfony4 with A2lixTranslationFormBundle and KnpDoctrine strategy example.

---

### Start:

- ```docker-compose build```
- ```docker-compose up```
- ```docker-compose exec app bin/console doctrine:schema:update --force```
- Go ```http://localhost:8000```
